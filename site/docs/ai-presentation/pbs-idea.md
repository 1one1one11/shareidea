# PB's Idea

이런 아이디어를 저는 가지고 있고 공유 드립니다.

## 아이디어 1: 이벤트 기반 상담 큐

- 고객 이벤트(만기/변동성/입출금) 발생 시 우선순위 자동 산정
- PB는 우선순위 높은 고객부터 대응

## 아이디어 2: 상담 전 1분 브리핑

- 고객별 핵심 이슈를 1분 요약 카드로 생성
- 통화/미팅 전 준비 시간을 단축

## 아이디어 3: 리서치 자동 큐레이션

- 고객 성향과 보유 자산에 맞춰 리포트 자동 매칭
- 불필요한 정보 노이즈 감소

## 실행 원칙

- 설명 가능한 로직
- 과도한 자동화보다 운영 일관성 우선
- PB 판단을 보조하고 대체하지 않음
