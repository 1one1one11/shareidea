# PB 고객관리 AI 시스템: 티어 기반 서비스 체계

티어 기반 서비스 차등화를 통해 상위 고객에 PB 리소스를 집중하고, 반복 업무는 자동화 채널로 처리한다.

## 문서 목적

- PB 서비스 티어 기준을 단순하고 일관되게 정의
- 티어별 서비스 범위/SLA를 명확히 규정
- 향후 CRM 대시보드 및 알림 시스템 확장 기준 제공

## Tier 1 - Platinum

자산 50억+ · 수익 상위 5% · Ultra HNW

- 포트폴리오: 실시간 모니터링 + 고객별 커스텀 대시보드
- 캐시플로우: 이자/배당/만기 이벤트 전량 추적 + 사전 알림
- 리서치: 당사 + 주요 외부 + 글로벌 소스 통합 요약
- 매매 지원: AI 시나리오 사전 생성 후 PB 검토 기반 제안
- 응대 SLA: 즉시 연결, 24시간 내 콜백 보장

## Tier 2 - Gold

자산 10~50억 · HNW

- 포트폴리오: 주간 리뷰 리포트 자동 생성 + PB 검수
- 캐시플로우: 주요 이자/배당 알림 제공
- 리서치: 당사 + 주요 외부 리서치 요약
- 매매 지원: 요청 기반 처리, 능동 제안 월 1~2회
- 응대 SLA: 1시간 내 콜백

## Tier 3 - Silver

자산 3~10억 · Mass Affluent

- 포트폴리오: 월간 리뷰 자동 생성/발송
- 캐시플로우: 만기/배당 기본 알림만 제공
- 리서치: 당사 리서치 요약 제공
- 매매 지원: AI 챗봇/RPA 우선 대응, PB 처리 지연 허용
- 응대 SLA: 2시간 내 콜백

## Tier 4 - Standard

자산 3억 미만 · General

- 포트폴리오: 분기 리포트 자동 발송
- 캐시플로우: 기본 미제공
- 리서치: 뉴스레터 수준 브리핑
- 매매 지원: MTS/HTS 유도, 단순 주문 자동화
- 응대 SLA: 당일 내 콜백(예약제)

## 단순업무 처리 프로토콜

핵심 원칙: `지연`이 아니라 `채널 분리`로 설계한다.

### 1) 디지털 채널 우선 안내

Tier 3~4의 단순 주문(시장가/정정/이체)은 AI ARS/챗봇 1차 응대로 처리하고, 모바일 주문 경로를 우선 안내한다.

### 2) 정보성 요청 자동 응답

잔고 조회, 세금 문의, 수익률 확인 같은 반복 조회성 요청은 전 티어 공통으로 자동 응답한다.

### 3) 콜백 큐 SLA 자동 관리

- Tier 1~2: 즉시 또는 1시간 내 응답
- Tier 3: 2시간 내 콜백
- Tier 4: 당일 내 콜백
